    **ANALYTICAL FORM OF F_vert(AoA) — document in docstring of get_foil_forces_aoa:**

    The per-vessel vertical force on an ascending vessel at tip-speed ratio lambda:
      v_tan = lambda * v_loop
      v_rel = v_loop * sqrt(1 + lambda^2)      [exact at fixed lambda]
      beta = arctan(v_loop / v_tan) = arctan(1/lambda)   [exact at fixed lambda]
      q = 0.5 * rho_w * A_foil * v_rel^2 = 0.5 * rho_w * A_foil * v_loop^2 * (1 + lambda^2)

      C_L_2D, C_D_0 = interpolate_naca(AoA_target_deg)   [fixed by AoA_target]
      C_L_3D = C_L_2D / (1 + 2/AR) = C_L_2D * AR / (AR + 2)
      C_D_i = C_L_3D^2 / (pi * e_oswald * AR)
      C_D_total = C_D_0 + C_D_i

      F_vert_per_vessel = -q * (C_L_3D * cos(beta) + C_D_total * sin(beta))
                        = -0.5 * rho_w * A_foil * v_loop^2 * (1 + lambda^2) *
                           (C_L_3D * cos(beta) + C_D_total * sin(beta))

    Since beta = arctan(1/lambda) is fixed at fixed lambda:
      F_vert scales as v_loop^2 for any fixed AoA_target.
      The sign is ALWAYS negative (downward) because C_L_3D > 0 and C_D_total > 0
      and cos(beta) > 0 and sin(beta) > 0 for beta in (0, 90 deg).

    The TOTAL vertical force opposing buoyancy (N_ascending vessels):
      F_vert_total = N_ascending * F_vert_per_vessel (negative = downward)
      |F_vert_total| = N_ascending * 0.5 * rho_w * A_foil * v_loop^2 * (1+lambda^2) *
                       (C_L_3D * cos(beta) + C_D_total * sin(beta))

    **Function get_foil_forces_aoa(v_loop, AoA_target_deg, lam=lambda_design):**

    CRITICAL CHANGE from Phase 4 sys01: In Phase 4, AoA_eff = beta - mount_angle (with
    mount_angle fixed at 38 deg). In Phase 5, AoA_target IS the free parameter. The mount
    angle is computed implicitly: mount_angle = beta - AoA_target (this sets the foil
    orientation so that AoA_eff = AoA_target at the current beta).

    Implementation:
      1. v_tan = lam * v_loop
      2. v_rel = sqrt(v_loop^2 + v_tan^2)
      3. beta_rad = atan2(v_loop, v_tan)  [arctan(v_loop/v_tan); angle from horizontal]
      4. beta_deg = degrees(beta_rad)
      5. alpha_clamped = max(0, min(14, AoA_target_deg))
         [Use AoA_target directly — it IS the effective AoA by construction]
      6. C_L_2D, C_D_0 = interpolate_naca(alpha_clamped)
      7. C_L_3D = C_L_2D / (1 + 2/foil_AR)
      8. C_D_i = C_L_3D^2 / (pi * e_oswald * foil_AR)
      9. C_D_total = C_D_0 + C_D_i
      10. q = 0.5 * rho_w * v_rel^2 * A_foil
      11. L_force = q * C_L_3D  [magnitude]
      12. D_force = q * C_D_total  [magnitude]
      13. F_tan  = L_force * sin(beta_rad) - D_force * cos(beta_rad)  [Phase 2 convention]
      14. F_vert = -L_force * cos(beta_rad) - D_force * sin(beta_rad) [Phase 2 convention; < 0]
      15. mount_angle_implied = beta_deg - AoA_target_deg  [diagnostic only]
      Return: F_tan, F_vert, L_force, D_force, v_rel, beta_deg, C_L_3D, C_D_total, mount_angle_implied

    Note: at AoA_target=10.0128 and lambda=0.9, beta_rad = arctan(1/0.9) = arctan(1.1111) =
    48.01 deg (approximately), consistent with Phase 4 beta_final_deg = 48.0128.

    **Function F_net_residual(v_loop, AoA_target_deg, lam):**
    F_vert_pv = get_foil_forces_aoa(v_loop, AoA_target_deg, lam)[1]  [negative]
    F_vert_total = N_ascending * F_vert_pv                             [negative]
    F_drag_hull = 0.5 * rho_w * C_D_hull * A_frontal * v_loop^2      [positive]
    F_net = F_b_avg_N + F_vert_total - F_drag_hull                    [= 0 at equilibrium]
    Return F_net  [positive when v_loop too low, negative when too high]
