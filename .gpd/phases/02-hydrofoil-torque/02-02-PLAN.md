          CAREFUL: at the diametrically opposite position of the same arm, the tangential
          direction reverses. But the ARM ROTATION continues in the same direction.
          At position D (180° from A on the same arm circle):
          The arm tip moves in the -x direction (tangential velocity direction reverses).
          v_vessel_D = (-v_tangential, 0, -v_loop)  [tangential -x, vertical down]
          v_flow_D   = (+v_tangential, 0, +v_loop)  [flow from forward-and-above in the +x, +z quadrant]

          WITHOUT tack: foil at same angle as ascending → flow attacks from wrong side → drag-dominated.
          WITH tack (foil flipped about span axis): foil leading edge faces (+v_tangential, 0, +v_loop).
          AoA_eff_D = beta_D = arctan(v_loop/v_tangential) [same magnitude as ascending by symmetry]
          NACA 0012: C_L(AoA_eff_D) = same magnitude as ascending (symmetric profile).
          Lift direction after tack: perpendicular to v_flow_D in the x-z plane.
          v_flow_D direction unit vector: (v_tangential, 0, v_loop)/v_rel
          Perpendicular (rotated 90° toward -x to oppose the arm's -x motion): wait, need arm rotation direction.

          The torque sign on the shaft is what matters. At position D, the arm tip is moving in the -x direction.
          For the arm to continue rotating CCW (viewed from above), we need a force in the -x direction at position D.
          The foil lift after tacking: perpendicular to v_flow_D = (+v_tan, 0, +v_loop).
          Rotating the flow vector 90° toward -x: L direction = (-v_loop, 0, +v_tan)/v_rel.
          F_tan_D (tangential component of lift, in the -x direction) = L * v_loop/v_rel.
          Since v_loop > 0, F_tan_D_x component = -L*v_loop/v_rel < 0 (in -x direction).
          Force in -x at position D drives CCW rotation (SAME rotational direction as +x at position A).
          Torque contribution: |F_tan_D| * r = L*v_loop/v_rel * r > 0 (same sign as ascending torque).

          Also write F_tan_D using the beta formula:
          F_tan_D = L*sin(beta_D) - D*cos(beta_D) [same formula as ascending, same beta magnitude]

          Both give positive torque contribution. Report the vector derivation explicitly.

    Step 2 — Rotating-arm tacking sign derivation (MANDATORY; not assumed by symmetry):

    ```python
    """
    Global coordinate frame (rotating-arm):
    +x: direction of arm tip velocity at the reference position (ascending vessel arm position)
    +z: vertical up
    Positive shaft rotation: CCW viewed from above → arm tip moves in +x at ascending position
                             and in -x at the diametrically opposite (descending) position.

    KEY: the arm is a SOLID ROTATING ARM. At any given moment, one arm tip is at
    the ascending-side reference position (velocity +x), and a DIFFERENT portion of the
    same arm (or different arm) is at the descending-side position. For a 3-arm machine,
    the arm at the descending position is a different arm, but the geometry is the same.

    Position A (ascending-side arm tip):
      Arm tip velocity: v_tip = (+v_tangential, 0, 0)  [moves in +x]
      Vessel velocity:  v_vessel_A = (+v_tangential, 0, +v_loop)  [arm tip + vessel ascent]
      Flow relative to foil: v_flow_A = -v_vessel_A = (-v_tan, 0, -v_loop)  [from forward-below]
      beta_A = arctan(v_loop/v_tan) from horizontal  [flow arrives from below-and-ahead]
      Mount angle set so AoA_eff = AoA_target at design lambda.
      Lift L_A: perpendicular to v_flow_A, directed toward +x (drives CCW rotation).
      F_tan_A = L_A * sin(beta_A) - D_A * cos(beta_A)  > 0 (from Plan 01).
      Torque_A = F_tan_A * r  > 0 (drives CCW).

    Position D (descending-side arm, 180° from A in arm circle):
      Arm tip velocity: v_tip = (-v_tangential, 0, 0)  [moves in -x, still CCW rotation]
      Vessel velocity:  v_vessel_D = (-v_tangential, 0, -v_loop)  [tip in -x + vessel descends]
      Flow relative to foil: v_flow_D = -v_vessel_D = (+v_tan, 0, +v_loop)  [from ahead-and-above in +x,+z]
      beta_D = arctan(v_loop/v_tan) from horizontal  [same angle, flow from above-and-ahead]

      WITHOUT tack: foil at same orientation as ascending side.
        Flow from (+v_tan, 0, +v_loop) hits the BACK of the foil (suction side is facing wrong way).
        Lift direction would be toward -x → torque negative → opposes rotation.

      WITH tack (foil flipped 180° about span axis = y-axis):
        Foil leading edge now faces the incoming flow (+v_tan, 0, +v_loop).
        NACA 0012 C_L(AoA_eff) = same magnitude by profile symmetry.
        Flow direction unit vector: n_flow_D = (+v_tan, 0, +v_loop) / v_rel

        Lift is perpendicular to n_flow_D in the x-z plane.
        Two perpendicular directions: rotate n_flow_D by 90°:
          Option 1: (-v_loop, 0, +v_tan) / v_rel  [points in -x, +z direction]
          Option 2: (+v_loop, 0, -v_tan) / v_rel  [points in +x, -z direction]

        The tack angle (foil orientation) determines which perpendicular.
        After tack, positive AoA gives lift in the direction that produces torque in the rotation direction.
        At position D, the rotation direction requires force in -x.
        The -x component of Option 1: (-v_loop/v_rel) < 0 → this is the -x direction. ✓
        The -x component of Option 2: (+v_loop/v_rel) > 0 → this gives +x direction. ✗

        The correct tack: lift direction = (-v_loop, 0, +v_tan) / v_rel.
        Tangential component (in -x direction, which drives CCW at position D):
          F_tan_D = L_D * v_loop/v_rel  [magnitude of -x component of lift]

        But also include drag contribution to tangential:
          Drag opposes v_flow_D direction: drag force = D * n_flow_D = D*(+v_tan, 0, +v_loop)/v_rel
          Tangential component of drag (in -x direction): D * (-v_tan/v_rel) ...
          Wait: drag is in the direction of v_flow_D = (+v_tan, 0, +v_loop)/v_rel.
          The -x component of drag = D * (+v_tan/v_rel) * (-1 for the -x component)...
          Actually drag force = D_mag * (+v_tan, 0, +v_loop)/v_rel.
          The x-component of drag = +D_mag * v_tan/v_rel (positive x → opposes -x CCW at D).

        So: net tangential force at D (in -x direction, driving CCW):
          F_tan_D = L_D * v_loop/v_rel - D_D * v_tan/v_rel
                  = (L_D * sin(beta_D) - D_D * cos(beta_D))  [since sin(beta)=v_loop/v_rel, cos(beta)=v_tan/v_rel]
                  = F_tan formula with same beta as ascending!

        This confirms: F_tan_D = L*sin(beta) - D*cos(beta) > 0 (same formula as ascending).
        Torque_D = F_tan_D * r  > 0 (drives CCW rotation — CONFIRMED).

    Key result: The tacking vector geometry gives F_tan_D = F_tan_A at the same |lambda|.
    The sign of the torque contribution is POSITIVE for both positions. QED.

    Darrieus cross-check: in VAWT notation, C_T = C_L*sin(phi) - C_D*cos(phi) > 0 when L/D > cot(phi).
    Here phi = beta = arctan(v_loop/v_tan). This is identical for both A and D. Consistent. ✓
    """

    # Implement the derivation numerically
    def descending_forces_after_tack(lambda_val, v_loop, AR, e, A_foil, naca_data, mount_angle_rad):
        """
        After tack-flip, flow is from (+v_tan, 0, +v_loop) direction.
        AoA_eff_dn = beta_D - mount_angle_rad (same formula as ascending by derivation above).
        Forces computed using same NACA table and Prandtl LL as ascending.
        """
        v_tan = lambda_val * v_loop
        v_rel = np.sqrt(v_loop**2 + v_tan**2)
        beta_rad = np.arctan2(v_loop, v_tan)

        AoA_eff_dn_deg = np.degrees(beta_rad - mount_angle_rad)
        CL_2D, CD_0 = interpolate_naca(np.clip(AoA_eff_dn_deg, 0, 14), naca_data)
        CL_3D = CL_2D / (1 + 2.0/AR)
        CD_i = CL_3D**2 / (np.pi * e * AR)
        CD_total = CD_0 + CD_i

        q = 0.5 * rho_w * v_rel**2 * A_foil
        L = q * CL_3D
        D = q * CD_total

        # Force decomposition (identical formula to ascending — derived above)
        F_tan_dn = L * np.sin(beta_rad) - D * np.cos(beta_rad)
        F_vert_dn = -L * np.cos(beta_rad) - D * np.sin(beta_rad)
        P_shaft_dn = F_tan_dn * v_tan

        return {"F_tan_dn_N": F_tan_dn, "F_vert_dn_N": F_vert_dn,
                "P_shaft_dn_W": P_shaft_dn, "L_N": L, "D_N": D,
                "CL_3D": CL_3D, "CD_total": CD_total,
                "beta_deg": np.degrees(beta_rad), "AoA_eff_deg": AoA_eff_dn_deg}
    ```
